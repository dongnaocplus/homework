# homework
该工程是学生的作业系统，管理和提交方法如下：
  
1. 班主任依据课程内容创建目录，格式可以为vip_num_课程名（num指课程次号）；
  
2. 学员在班主任创建的对应目录里提交作业，但是提交时以学员名称（QQ名称）命名的目录里；
